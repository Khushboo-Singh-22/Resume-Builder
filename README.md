# 🧠 AI Resume Builder (MERN Stack)

A full-stack AI-powered Resume Builder web application that helps users create professional, ATS-friendly resumes in minutes.

---

## 🚀 Live Demo

👉 https://resume-builder-caqw.vercel.app

---

## ✨ Features

* 📄 Create resume using simple forms
* 🎨 Multiple resume templates (Classic, Modern, Minimal, Tech Resume)
* 👀 Live resume preview
* 📥 Download resume as PDF
* 🤖 AI Resume Enhancement (better wording & suggestions)
* 📊 ATS Resume Checker (score + keyword suggestions)
* 🔐 User Authentication (Login / Signup)
* ☁️ Cloud-based storage

---

## 🛠️ Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS
* Redux Toolkit
* Firebase Authentication
* html2canvas + jsPDF

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication

### Tools & Services

* Git & GitHub
* Vercel (Deployment)
* Firebase

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/Khushboo-Singh-22/Resume-Builder.git
cd Resume-Builder
```

---

### 2. Frontend Setup

```bash
cd Client
npm install
npm run dev
```

---

### 3. Backend Setup

```bash
cd Server
npm install
npm run server
```

---

## 🔐 Environment Variables

Create a `.env` file inside Server:

```
MONGO_URL=your_mongodb_connection
JWT_SECRET=your_secret_key
```
